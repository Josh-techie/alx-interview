# UTF-8 Validation

## Overview

This project, part of [ALX Africa](https://www.alxafrica.com/) Intranet's Short Specializations Average, it involves implementing a Python method that determines whether a given data set represents a valid UTF-8 encoding.

---

## Project Structure

- **Directory:** 0x04. UTF-8 Validation
- **File:** [0-validate_utf8.py](./0-validate_utf8.py)

## Tasks

**Create a method validUTF8(data):**

- Returns True if the given data is a valid UTF-8 encoding.
- Returns False otherwise.
- A character in UTF-8 can be 1 to 4 bytes long.
- The data set can contain multiple characters.
- The data is represented by a list of integers, where each integer represents 1 byte of data, and only the 8 least significant bits need to be handled.

---

## How to Run

Execute `./0-main.py` to test your `validUTF8` method.

```bash
$ ./0-main.py
Valid UTF-8 encoding: True
Valid UTF-8 encoding: True
Valid UTF-8 encoding: False
```
