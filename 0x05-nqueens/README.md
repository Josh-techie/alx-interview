<h1> 0x05. N Queens </h1>

- This project, part of [ALX Africa](https://www.alxafrica.com/) Intranet's Short Specializations Average, involves solving the N queens puzzle. The challenge is to place N non-attacking queens on an N×N chessboard. The goal is to write a Python program that efficiently solves the N queens problem.

<h1> Task</h1>

- Chess grandmaster Judit Polgár, the strongest female chess player of all time, inspired this task. The N queens puzzle involves placing N non-attacking queens on an N×N chessboard. The Python program should print every possible solution to the problem, with one solution per line.

---

<h1> Usage </h1>

```bash
$ ./0-nqueens.py 4
[[0, 1], [1, 3], [2, 0], [3, 2]]
[[0, 2], [1, 0], [2, 3], [3, 1]]
```
